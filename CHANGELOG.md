# 更新日志

## v1.0.0 - 2026-07-09

### 新增

- 完成网页端训练检测、动作分析、训练报告、AI 教练、训练历史和系统设置页面。
- 支持视频上传后调用 `/api/analyze` 进行姿态识别、动作分类、击球事件检测和训练报告生成。
- 支持标注视频生成，展示人体骨架、动作标签和击球事件。
- 支持半场步伐热力图和目标球员移动轨迹。
- 支持逐拍时间轴，点击击球事件可跳转到对应视频片段。
- 支持 AI 教练问答，预留 Qwen 和智谱 AI 接口。
- 支持网页账号、微信登录绑定、训练历史同步。
- 新增微信小程序端，包括训练、分析、AI 教练、我的、网页账号绑定等页面。
- 新增多语言界面：中文、English、日本語、한국어、Bahasa Indonesia。
- 新增 Windows 一键启动脚本 `start_web.bat`。

### 优化

- YOLO-Pose 模型缓存，避免重复加载模型。
- 同视频同配置复用标注视频，减少重复分析耗时。
- 比赛远景模式使用更适合长视频的采样和渲染策略。
- 标注视频支持预览帧率输出，缩短视频生成时间。
- 默认关闭逐拍短片导出，减少无效转码。
- 前端使用 SSE 展示分析进度。
- AI 教练回答做格式清洗，减少 Markdown 占位符。
- 步伐热力图更新为半场热区展示，增强产品可读性。

### 已知限制

- 动作分类效果依赖训练数据规模和拍摄角度。
- 比赛远景视频中人物过小或遮挡严重时，姿态识别和目标跟踪会下降。
- CPU 环境可运行，但视频分析耗时明显高于 GPU 环境。
- AI 教练需要配置第三方大模型 API Key。
